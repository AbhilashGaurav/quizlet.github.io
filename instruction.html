Instruction cum code of conduct 

01] After selecting the subject of the quiz, the user will be shown  
    10 questions one question at a time on screen.The next question 
    will be shown only when the current question is marked and once
    the question is marked the user cannot go back.

02] Clock timer will run on top and the maximum timing is given to 
    the player after which the test will be submited.

03] Once a user marks the answer the correct answer tile will be 
    Blue in colour and the wrong answer tile will be Red in colour.

04] If the user skipped one question and entered the next button 
    then the user will be allowed to select 2 options in the next
    question and vice versa.
    
05] At the end number of correct questions and the time will be 
    recorded and this will be the criteria of ranking of the leader
    board.
